# **App Name**: Event Horizon Planner

## Core Features:

- Blog Management: Enable CRUD operations for blog posts, allowing admin users to create, read, update, and delete content.
- Vendor Directory: Create and manage a vendor directory, enabling the display, filtering, and updating of vendor profiles by administrators.
- Budget Calculator: Implement a tool that estimates event costs and provides a downloadable summary for users.
- RSVP Tracker: Provide a feature to manage guest lists with status updates, assisting users in tracking attendance.
- Event Checklist: Offer an interactive task list for event preparation, helping users organize and track their progress.
- AI Event Theme Generator: Utilize AI to suggest event themes and color palettes based on user input, functioning as a tool to enhance the event planning process.
- Contact Form: Provide a form to manage guest lists with status updates, assisting users in tracking attendance.

## Style Guidelines:

- Primary color: Rich aubergine (#430A4A) for a sophisticated, yet modern feel, reflecting elegance and creativity.
- Background color: Light grey (#F0F0F0), to create a neutral backdrop that enhances readability and visual balance.
- Accent color: Coral (#FF7F50), which is analogous to aubergine, adding a lively, eye-catching contrast for key interactive elements.
- Headline font: 'Playfair', serif, for titles to convey elegance, using 'PT Sans', sans-serif, for body, for readability.
- Use clean, minimalist icons to represent different event categories and tools. Consistent in style, in a rounded style.
- Employ a grid-based layout for content to ensure responsiveness across devices, emphasizing a clean and structured design.
- Subtle animations such as fading transitions and smooth scrolling will be implemented to enhance user experience.
